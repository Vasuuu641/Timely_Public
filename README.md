# Timely

**Timely** is a full-stack productivity app designed to help students and professionals manage tasks, schedules, notes, daily reviews, and Pomodoro sessions efficiently. The project is built using TypeScript, Node.js, NestJS, and Vite for the frontend.

## Tech Stack
- **Frontend:** TypeScript, React, Vite, CSS
- **Backend:** Node.js, NestJS, Prisma, MySQL
- **Tools & Libraries:** Docker, ESLint, Prettier, Git
- **Other:** JWT for authentication, REST API

## Features
- **Task Management:** Add, update, and delete tasks
- **Pomodoro Timer:** Track focused work sessions with breaks
- **Daily Review:** Reflect on tasks completed and pending
- **Notes:** Create and organize notes efficiently
- **Schedule:** Manage events and deadlines
- **User Roles:** Admin and User roles with access control
