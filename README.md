# Timely

**Timely** is a student-centric productivity app designed to help students organize their day, track their progress, and stay focused. It combines essential productivity tools with gamified elements to make studying and task management engaging and effective.

---

## Features

Timely provides a suite of features tailored for students:

- **Pomodoro Timer (Gamified)** – Focus on tasks with adjustable difficulty levels (easy, medium, hard) and earn points as you complete sessions.  
- **Schedule Organizer** – Create, view, and manage your daily or weekly schedule.  
- **To-Do List** – Keep track of tasks and mark them as completed.  
- **Notes** – Save and organize notes for your subjects or personal projects.  
- **Daily Review** – Reflect on your day, track mood or productivity, and rate your day.  
- **Quiz** – Test yourself with quizzes to improve retention and learning.

---

## Tech Stack

Timely is built using modern web development technologies:

- **Frontend:** React + TypeScript + Vite  
- **Backend:** Node.js + NestJS + TypeScript  
- **Database:** MySQL (running in Docker) with Prisma ORM  
- **Authentication:** JWT (JSON Web Tokens)  
- **State Management:** React Context for user session management  
- **Styling & UI:** Plain CSS  
- **HTTP Requests:** Axios
